# Ai_Powered_Learning_Assistant
This project was designed to meet the need of a students who want to use ai in there day to day works and assignment. In this project a user can register using username and email and creadentials will be used to sign in and then user can use six different ai tool according to their requirement like Ai Note Generator , Ai- Doubt Solver, Ai-Quiz Etc
